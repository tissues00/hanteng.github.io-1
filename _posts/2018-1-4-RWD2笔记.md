### RWD2

##### 其实都是我的错题
```
<html> → 定义 HTML 文档, 

<p> → 定义段落,

<body> → 定义文档的主体, 

<h1> to <h6> → 定义 HTML 主体內标题,

<title> → 定义文档的标题, 

<!DOCTYPE> → 定义文档类型, 

<!--...--> → 定义注释, 

<br> → 定义简单的折行, 

<hr> → 定义水平线
```

```
<textarea> → 定义多行的文本输入控件, 
<input> → 定义输入控件, 
<datalist> → 定义下拉列表, 
<option> → 定义选择列表中的选项, 
<label> → 定义 input 元素的标注, 
<optgroup> → 定义选择列表中相关选项的组合, 
<form> → 定义供用户输入的 HTML 表单, 
<select> → 定义选择列表（下拉列表）, 
<button> → 定义按钮, 
<fieldset> → 定义围绕表单中元素的边框來组合表单中的相关元素
```

